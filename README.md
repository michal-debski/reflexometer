# Reflexometer

## Description
The Reflexometer project is designed to provide users with an advanced platform for measuring reflexes in various contexts. Whether it's for sports training, psychological assessments, or gaming scenarios, Reflexometer inspires individuals to improve their reflex actions.

## Features
- **Real-Time Reflex Measurement:** Capture and analyze reflex speed in real-time.
- **User-Friendly Interface:** An intuitive design that enables users to navigate and utilize the application effortlessly.
- **Data Visualization:** Display reflex measurements through interactive charts and graphs.
- **Customizable Tests:** Users can create customized tests to challenge and measure their reflexes effectively.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/michal-debski/reflexometer.git
   ```

2. Navigate to the project directory:
   ```bash
   cd reflexometer
   ```

3. Install the necessary dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```

## How to Use
- **Starting a Test:** Click on the 'Start Test' button to begin measuring your reflexes.
- **Viewing Results:** Once the test is complete, results will be displayed in the results section.
- **Historical Data:** Users can access their past reflex test results for analysis and tracking.
